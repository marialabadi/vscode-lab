
# Refactoring Project

## 1. Techniques Used

### Rename Method
- Renamed methods to meaningful names:
  - calculateSum
  - calculateProduct
  - calculateRatio
  - displayResult

### Extract Method
- Split one large method into smaller methods:
  - calculateSum()
  - calculateProduct()
  - calculateRatio()

## 2. Improvements

- Improved readability
- Reduced complexity
- Easier to maintain

## 3. Error Handling

- Used ArithmeticException to handle division by zero
